**Run locally on Jupyter Lab**

-Install Python 

Download from python.org. Important: Check the box "Add Python to PATH" during installation.

-Install Git 

Download from git-scm.com. This allows the git clone command to work.  

-Install VS Code 

Download from code.visualstudio.com.

-clone the repository

git clone https://github.com/claudio1975/Python_Crash_Course.git
cd Python_Crash_Course

-create the environment

python -m venv venv

-activate the environment

venv\Scripts\activate (Windows) or source venv/bin/activate (Mac/Linux)

-install jupyter lab

pip install jupyterlab

-launch jupyter lab

jupyter lab

-select the notebook to open
